Gitコミット貢献度グラフジェネレーター

コミットログを貼り付けて、貢献度の円グラフを生成します。

1 コミットログを取得

Windows環境では、以下のコマンドを実行すると結果が自動でクリップボードにコピーされます。

git log --pretty=format:'%an' | clip

分析したいリポジトリのフォルダで以下のコマンドを実行し、表示された結果をすべてコピーしてください。

git log --pretty=format:'%an'

2 ログを貼り付けてグラフを生成
